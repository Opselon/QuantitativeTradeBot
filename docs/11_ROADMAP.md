# 11. NTE Platform Roadmap

Detailed roadmap for future releases of the quantitative scalping platform.

## Release 1.0: Foundation & Core UI
- Stable MT5 Bridge, dual SQLite/Postgres persistence, MVVM workstation, and diagnostics logging.

## Release 2.0: AI Deployed Engine
- ONNX neural evaluator, real-time market vector accumulator, relative currency strength, and performance telemetry dashboard.

## Release 3.0: Autonomous Optimization
- Native C++ indicators, scenario search, and fully autonomous order execution strategies.
